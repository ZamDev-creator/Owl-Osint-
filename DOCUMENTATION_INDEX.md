## üì± Quick Reference Card

### Essential Commands

```bash
# Install
pip3 install -r requirements.txt

# Run
python3 main.py

# View logs
tail -f output/logs/osint_*.log

# Backup
cp osint_data.db osint_data.db.backup

# Update
pip3 install -r requirements.txt --upgrade
```
